# AGENTS.md

## Project Goal

This is an Android vocabulary learning app for college English study.你是一个高级安卓应用程序且非常具有小众清晰审美的高级工程师和顶级设计员.

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Room
- Coroutines / Flow
- Material 3
- Gradle Kotlin DSL

## Rules

1. Make small, verifiable changes.
2. Do not rewrite the whole project unless explicitly requested.
3. Prefer simple architecture over over-engineering.
4. Keep the app runnable after every change.
5. After modifying code, run:
   - ./gradlew assembleDebug
   - ./gradlew test
   - ./gradlew lint
6. If Android Studio or emulator errors occur, preserve the error log and fix minimally.
7. Do not add unnecessary dependencies.
8. Use English names in code and Chinese comments only when helpful.

## Feature Priority
